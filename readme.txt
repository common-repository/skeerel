Acceptez facilement les paiements avec Skeerel sur votre site WooCommerce.

== Description ==

[Skeerel](https://skeerel.com) est la solution d'achat en 1 clic qui booste votre taux de conversion.

* C'est **rapide** : vos clients passent commande depuis la page produit ou le panier en accéléré
* C'est **simple** : nous nous occupons de tout le parcours d'achat et vous transférons automatiquement les données de vos clients
* C'est **sécurisé** : à l'aide de notre système d'authentification breveté

https://www.youtube.com/watch?v=PxIjkFZaoyM&rel=0

Le module [Skeerel](https://skeerel.com) pour WooCommerce vous permet d'intégrer l'achat en 1 clic via Skeerel sur votre site marchand.

Envie de voir une démonstration ? C'est ici que ça se passe => [Démo Skeerel WooCommerce](https://woocommerce.skeerel.com)

= Pré-requis =

* Un compte [Skeerel e-commerce](https://admin.skeerel.com)
* Un site internet avec WordPress et WooCommerce installés

= Fonctionnalités =

* Achat en 1 clic via la page panier
* Achat en 1 clic via la page produit 
* Compatible avec la nouvelle règlementation européenne DSP2 et SCA
* Compatible avec ordinateur & smartphone
* Paiements par carte bancaire, Apple Pay ou Google Pay
* Tous vos modes de livraison compatibles
* Configuration simple
* Activation uniquement pour les administrateurs du site pour tester
* Activation du mode "Bac à sable" avant de passer en production
* Gestion des paiements reçus directement depuis le détail de la commande dans WooCommerce

== Installation ==

Une fois le module activé, vous êtes automatiquement redirigé vers la page d'onboarding.

1. Créez un compte [Skeerel e-commerce](https://admin.skeerel.com) (ou cliquez sur j'ai déjà un compte)
1. Si ce n'est pas déjà fait, [enregistrez votre site internet](https://admin.skeerel.com/websites) sur votre espace e-commerçant
1. Saisissez l'identifiant et la clé secrète du site créé sur votre compte Skeerel e-commerce
1. Configurez les options du module
1. C'est fini ! Acceptez les paiements via Skeerel

== Frequently Asked Questions ==

= Quels sont les moyens de paiements acceptés ? =

Cartes bancaires européennes, Google Pay, Apple Pay

= Quelles informations de l'utilisateur me sont retournées ? =

Toutes les informations nécessaire à la commande : 
* Adresse mail et profil acheteur
* Adresse de livraison (si nécessaire)
* Adresse de facturation
* Statut du paiement
* ...

= Est-ce que le numéro de carte bancaire m'est retourné ? =

Non ! 

= Est-ce que c'est payant ? =

Le module WordPress est gratuit. Il n'y a aucun frais d'installation, seulement une commission prise sur chaque paiement. Pour plus d'informations, [visitez la page tarifs](http://skeerel.com/tarifs/).
